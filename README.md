# Selenium Automation Testing Project

## Overview

This is a Selenium-based automation testing framework built with Java. The project demonstrates functional testing of web applications using the Page Object Model design pattern.

## Technologies Used

- Java
- Selenium WebDriver
- TestNG
- Maven

## Project Structure

seleniumAutomation/
├── src/test/java/
│ ├── pages/
│ └── tests/
├── pom.xml
├── testng.xml
└── README.md

## Prerequisites

- Java JDK 8 or higher
- Maven
- Chrome or Firefox browser

## Setup Instructions

1. Clone the repository:

git clone https://github.com/vasnaummer/seleniumAutomation.git

2. Navigate to project folder:

cd seleniumAutomation

3. Install dependencies:

mvn clean install

## How to Run Tests

Execute all tests:

mvn test

## Test Reports

After running tests, view the report at:

test-output/index.html
